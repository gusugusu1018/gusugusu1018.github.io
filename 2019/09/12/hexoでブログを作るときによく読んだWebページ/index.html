hexoでブログを作るときによく読んだWebページ
Created 2019-09-12| Updated 2019-09-12

hexoでブログを作るときによく読んだページを雑にまとめておくことにしました．
質問などあれば気軽にコメントしてください．

私が使用しているテーマ”Butterfly”製作者のjerrycさんのページ

  • https://jerryc.me/posts/21cfbf15/

めちゃくちゃお世話になっています．
中国語なので読むのは少し大変ですが，全部のっています．

hexoユーザはおそらく中国の方が多いので調べると中国語の記事ばかり出てくるのですが，TEAさんは日本語でhexo関連の記事を複数投稿しておりとても勉強になります．
TEAさんが使用しているhexoのプラグイン

  • https://photo-tea.com/p/usecase-of-hexo-plugin/

TEAさんがwordpressからhexoに移行した理由など…

  • https://photo-tea.com/p/wordpress-to-hexo/

こちらのWebページは2019年9月現在Github pagesでホスティングしているのですが，Netlifyという無料サーバがおすすめみたいですね．
コンテンツを増やして，順調にPVが増えたら移行してみようかなと思っています．

Qiitaのhexoタグ

  • https://qiita.com/tags/hexo

暇なときにhexo記事を読んでいます．

QracさんのHexoサイトをGitHubでバックアップ取りつつGitHub Pagesで公開もする方法

  • https://tech.qookie.jp/posts/hexo-deploy-github-pages-backup-version/

誰しもがやりたいと思う手法を公開してくださっていました．
しかし，初めの方は参考にしていたのだがgh-pagesであげてしまうと，ユーザページへの設定ができなくなるという問題があり，私は最終的に2つのリポジトリに分けることにしました．

結果的に，API Keyなどが含まれるソースのリポジトリを非公開にしたので，これでよかったんだと思います．
正直ブランチ分けてソースとサイトをアップするより，リポジトリ分けるほうがおすすめです．

Qracさんは先ほど挙げたNetlifyにデプロイする記事も上げてくださっています．

  • https://tech.qookie.jp/posts/hexo-deploy-netlify-github-backup-version/
  • https://tech.qookie.jp/posts/hexo-deploy-esa-netlify-github-backup-version/

とても参考になりますね！

謎が深いmachine powersさんのhexo記事もよく拝見しています．
OCRについての記事もあり，気になるところです．

  • https://blog.machine-powers.net/2018/06/25/hexo-with-netlify/
  • https://blog.machine-powers.net/2018/06/26/setup-hexo-detail/

以上ですが，まだまだブログは発展途上なので随時リンクを追加していくと思います．

それでは徒然文を最後まで読んでいただきありがとうございました．
